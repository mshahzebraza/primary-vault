### 🔧 Steps to Create a Bash Alias

1. **Open `.bashrc` for editing**: `cursor ~/.bashrc`
2. **Add alias** (Linux-style path): `alias dev='cd /f/dev/work/'`
3. **Save and exit**
4. **Apply changes**: `source ~/.bashrc`

### ✅ Usage

Now you can simply run `dev` to `cd` into your project folder.
