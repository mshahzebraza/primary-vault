## Overview
> Video Title: How I Rank #1 on Google with Claude Code 
 
Jesse Cunningham demonstrates building and optimizing a local business website using Claude Code — integrating real-world search data and automating SEO strategy. The shift: from manual site building to a data-driven, agentic approach.

Primary interest: learn the SEO perspective and use AI to fill in that gap as a developer.

---

## Key Concepts

### Website Hub Structure (0:47)

Four primary hubs that organize the site for both users and Google:

| Hub | Purpose |
|-----|---------|
| **Services** | Defines what the business offers |
| **Service Areas** | Templated location-based pages; tells Google where the business operates |
| **Trust & Company** | About Us + reviews consolidated |
| **Conversions** | Call-to-action optimization |

### Data-Driven SEO via DataForSEO API (2:34)

- Pulls real-world search volume and competitor keyword data
- Avoids expensive subscriptions (Semrush, Ahrefs) — custom API calls cost pennies
- "Guard the money matrix": only build pages for niches with actual search demand
- Example finding: roof *repair* outranks roof *replacement* in search demand → prioritize repair pages

### Automated Observe → Diagnose → Act Loop (8:43)

- APIs identify high-performing, high-intent keywords automatically
- Claude Code drives the automation — reads data, generates page content, prioritizes builds
- Scales content production without proportional manual effort
